## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# generate static project
$ npm run generate
# then copy the content of the "dist" folder to webserver
# tested with nginx 1.14.0 on windows 7
```
